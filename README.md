# Muasub Landing Page

## Cấu trúc
- `index.html`: file HTML chính
- `assets/css/style.css`: CSS đã được tách riêng

## Đưa lên GitHub Pages
1. Tạo repository mới trên GitHub.
2. Upload toàn bộ các file/thư mục trong gói này.
3. Vào Settings → Pages.
4. Chọn Branch: `main`, Folder: `/root`.
5. Bấm Save và chờ GitHub tạo link website.
